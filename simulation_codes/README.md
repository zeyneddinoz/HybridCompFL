## 💻 Simulation Environment

To run simulation experiments and visualize results, install the required packages using Python (version 3.10 or higher):

```python
pip install -r requirements.txt
```

No Raspberry Pi? No problem! This simulation environment allows you to evaluate models, datasets, and hyperparameters directly on your workstation.

### 🚀 How to Run
1. **Configure:** Open `main.py` and modify the settings (model selection, dataset, and hyperparameters) to suit your experiment.
2. **Execute:** Run the script using:
   ```bash
   python main.py
